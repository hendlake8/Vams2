# 게임 컨셉 브리프: Vams2

## 컨셉 요약

| 항목 | 내용 |
|------|------|
| 한 줄 요약 | 판타지 세계관의 모바일 싱글 뱀서라이크 |
| 핵심 재미 | 자동공격 + 스킬 빌드 조합 + 대량 몬스터 섬멸 쾌감 |
| 타겟 유저 | 캐주얼~미드코어 모바일 유저, 짧은 세션(5분)으로 반복 플레이 |
| 레퍼런스 | 탕탕특공대(Survivor.io)의 코어 루프를 충실히 재현 |
| 플랫폼 | 모바일 (Android/iOS) |
| 개발 규모 | 1인 개발, MVP 우선 |
| 수익 모델 | 미정 |

## 코어 루프

```
입장 → 5분 생존 (이동+자동공격+레벨업 스킬선택+스킬진화)
→ 사망 or 클리어 → 결과 정산/재화 획득 → 로비
→ 장비 강화/교체 → 재입장
```

## MVP 시스템

### 조작

- 플로팅 싱글 조이스틱 (터치한 곳에서 생성)
- 공격은 완전 자동, 가장 가까운 적 타겟팅

### 스킬 시스템

레벨업 시 3가지 스킬 중 1개 선택. 동일 스킬 재선택 시 레벨업 (최대 5).

**액티브 스킬 (6종, 최대 6슬롯):**

| 스킬 | 패턴 | 특성 |
|------|------|------|
| 매직 볼트 | 직선탄 | 기본 무기 업그레이드형 |
| 회전 검 | 캐릭터 주변 회전 | 근접 방어 |
| 파이어볼 | 폭발 범위 | 고데미지 |
| 아이스 스피어 | 관통 | 감속 부여 |
| 라이트닝 | 체인 | 다수 타겟 |
| 포이즌 클라우드 | 장판 | 지속 데미지 |

**패시브 스킬 (4종, 최대 6슬롯):**

| 스킬 | 효과 |
|------|------|
| 마나 서지 | 공격속도 증가 |
| 원소 핵 | 원소 데미지 증가 |
| 마법 갑옷 | 방어력 + 넉백 저항 |
| 신속의 부츠 | 이동속도 + 회피 |

### 스킬 진화 (돌파)

특정 액티브 + 패시브 조합이 모두 최대 레벨일 때 진화 가능.

| 액티브 | 패시브 | 진화 결과 |
|--------|--------|-----------|
| 파이어볼 | 원소 핵 | 메테오 스트라이크 |
| 라이트닝 | 마나 서지 | 썬더 스톰 |
| 회전 검 | 마법 갑옷 | 블레이드 배리어 |

진화 시 데미지 2배 + 시각 이펙트 대폭 강화.

### 맵

- 타일 기반 무한 반복 맵 (플레이어 중심으로 주변 타일 재배치)
- 판타지 숲/초원 테마
- 보스 등장 시 화면 경계에 원형 벽 생성 (보스 아레나)

### 웨이브 페이싱 (5분 기준)

| 시간 | 내용 |
|------|------|
| 0:00~0:50 | 슬라임 계열 (느림, 체력 낮음) — 온보딩 |
| 0:50~1:30 | 늑대 추가 (빠름) — 이동 회피 학습 |
| 1:30 | ★ 미니보스 1 |
| 1:30~2:30 | 스켈레톤 추가 (원거리 공격) — 스킬 빌드 중요 구간 |
| 2:30 | ★ 미니보스 2 |
| 2:30~4:00 | 혼합 웨이브, 밀도 급증 — 스킬 진화 발동 타이밍 |
| 4:00 | ★ 미니보스 3 |
| 4:00~4:30 | 최고 밀도, 엘리트 몬스터 — 빌드 완성도 테스트 |
| 4:30~5:00 | ★ 최종 보스 |

### 세션 종료

- 사망 또는 클리어 시 결과 정산 화면 표시
- 획득 재화 표시 후 로비 복귀
- 재시작 버튼 없음 (반드시 로비 경유)

### 캐릭터

- MVP에서는 기본 캐릭터 1명만 제공

## 수치 프레임워크

| 항목 | 공식 | 비고 |
|------|------|------|
| 기본 공격력 | `BaseDmg × (1 + SkillLv × 0.2)` | 스킬 레벨당 20% 증가 |
| 적 체력 | `BaseHP × (1 + WaveTime × 0.15)` | 시간당 15% 증가 |
| 경험치 요구량 | `10 + (Lv × 5)` | 레벨당 선형 증가 |
| 크리티컬 확률 | 기본 5% | 패시브로 증가 가능 |
| 크리티컬 배율 | 1.5x | 패시브로 증가 가능 |

## MVP 마일스톤

```
M1: 이동(플로팅 조이스틱) + 자동공격 + 적 스폰
M2: 경험치 + 레벨업 스킬선택 UI (3카드)
M3: 스킬 6종 + 패시브 4종 구현
M4: 스킬 진화 3종 + 보스
M5: 5분 세션 완성 (웨이브 테이블 + 결과/로비)
```

## MVP 이후 확장 계획

### 1순위: 장비 시스템

- 로비에서 장비 장착/강화
- 인게임 스탯 반영 (공격력, 체력, 이속 등)
- 장비 드롭 → 수집 → 합성 루프

### 후순위 (미확정)

- 다중 캐릭터 선택
- 챕터/스테이지 시스템
- 추가 맵 테마
- 수익 모델 적용

## 아트 방향

- 2D 탑다운 판타지
- 에셋 스토어 활용 필수 (1인 개발)
- 판타지 2D 탑다운 에셋 팩 확보 후 톤 통일

## 레퍼런스 분석: 탕탕특공대와의 차이

| 항목 | 탕탕특공대 | Vams2 |
|------|-----------|-------|
| 온라인 | 협동, 랭킹, 시즌패스 | 완전 싱글 |
| 세션 | 8분 / 15분 | 5분 |
| 테마 | 현대 도시 + 좀비 | 판타지 |
| 수익 모델 | F2P (광고+IAP) | 미정 |
| 캐릭터 | 다수 (유료 해금) | MVP 1명 |
| 장비 | 초기부터 존재 | MVP 이후 추가 |
