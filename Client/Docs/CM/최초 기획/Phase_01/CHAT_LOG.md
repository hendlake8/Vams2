## 탕탕특공대 (Survivor.io) 게임 디자인 요소 분석

### 1. 코어 게임플레이 메커닉

- **장르**: F2P 2D 로그라이크 서바이벌 (뱀파이어 서바이버즈 라이크)
- **조작**: 싱글 스틱 이동만 담당하고, 공격은 완전 자동. 이동과 회피에 집중하는 구조
- **세션 구조**: 한 판당 8분(유한맵) 또는 15분(무한맵/세로맵)의 제한 시간 동안 생존
- **레벨업 시스템**: 몬스터 처치 시 드롭되는 보석을 수집하면 경험치 게이지가 차고, 레벨업 시 3가지 스킬 중 하나를 선택
- **보스전**: 뱀파이어 서바이버즈와 달리 보스 등장 시 별도 아레나가 형성되어 본격적인 보스 패턴 대응이 필요 (적 웨이브 사이에 명확한 보스 페이즈가 존재)
- **맵 인터랙션**: 상자 파괴 시 소프트 통화, 회복 고기, 적 소탕 폭탄, 경험치 자석 등 획득

### 2. 캐릭터/영웅 시스템

버전 1.8부터 도입된 다중 캐릭터 선택 시스템이다.

- **기본 캐릭터**: 무료 기본 캐릭터 제공
- **잠긴 캐릭터**: 약 8종 이상 존재 (지속 추가 중)
- **고유 능력 예시**:
  - **Tsukiyomi**: 월영참 (Moonshade Slash) - 강력한 근접 범위 공격
  - **King**: 크리티컬 확률/데미지 특화 - 보스전 극대화
  - **Master Yang**: 스탯 스케일링 최강 - 범용 최상위
  - **Catnips**: 메디 드론 액티브 스킬 - 힐링 존 생성
- **해금 방법**:
  - 시즌 패스 구매 (약 $19.99)
  - 히어로 샤드 수집 (한정 이벤트, Ender's Echo 상자, Special Ops 코인 교환)
  - 직접 구매 (약 $29.99/캐릭터)
- **성장**: Energy Essence(퀵 패트롤에서 획득)를 사용해 레벨업

### 3. 스킬 시스템

인게임 스킬은 크게 두 종류로 나뉜다.

- **액티브 스킬 (무기)**: 실제 공격을 수행하는 스킬. 최대 6개 장착
- **패시브 스킬 (지원품)**: 버프 효과를 제공. 최대 6개 장착
- **돌파(진화) 조합**: 핵심 시스템. 특정 무기 + 특정 지원품 조합으로 무기가 5성 이후 진화 가능
  - 축구공 + 운동화
  - 번개 발사기 + 에너지 큐브
  - 드릴샷 + 탄약 추진기
  - 화염병 + 석유채
  - 부메랑 + 자석
- **티어 평가**:
  - S티어: 축구공, 번개 발사기, 드릴샷
  - A티어: 로켓 발사기, 지뢰
  - B티어 이하: 레이저, 두리안, 부메랑, 보호막, 화염병

### 4. 장비 시스템

인게임 스킬과 별개로, 메타 레이어에서 영구 장비를 관리한다.

- **장비 슬롯**: 무기, 방어구, 액세서리 등
- **등급 체계** (6단계):
  - 일반(회색) -> 우수(초록) -> 레어(파랑) -> 엘리트(보라) -> 에픽(노랑) -> 레전드(빨강)
- **S등급 장비**: 엘리트 이상에서만 존재하는 특수 등급
- **합성 시스템**: 동일 장비 합성으로 등급 상승. 레전드 1개 제작에 에픽 3개(= 엘리트 6개) 필요
- **등급 스킬**: 등급이 오를수록 추가 능력 해금 (무기는 우수부터, 방어구는 레어부터)
- **신의 주조**: 레전드 도달 후 추가 강화 시스템 (3단계)
  - 예: 빛을 쫓는 자 - 1단계 검열 추가, 2단계 공격력 +20%, 3단계 거대검술
- **무기-지원품 테마 연동**: 장검류는 부랑자의 갑옷, 총기류는 고화력 탄약, 쿠나이는 닌자 스크롤 등 직관적 조합

### 5. 진행 시스템

- **챕터 시스템**: 메인 스테이지. 3종류 맵 형식 (무한맵, 세로맵, 유한맵)
- **도전 모드**: 챕터 2 클리어 후 해금. 강화된 적 + 향상된 보상
- **시련의 길**: 단계별 돌파 도전
- **압도 모드**: 모든 돌파가 완료된 상태로 시작하는 특수 모드 (2024.10 추가)
- **퀵 패트롤**: 자동 전투로 자원 수급
- **시즌 패스**: 티어 진행에 따른 보상 (히어로 샤드 포함)

### 6. 수익화 모델

출시 3년 후에도 월 약 500만 달러의 매출을 유지하는 수익화 구조이다.

- **점진적 해금 방식**: 레벨 1에서 모든 과금 요소를 노출하지 않고, 게임 루프와 함께 하나씩 순차 해금하여 자연스러운 과금 유도
- **보상형 광고**: 상점에서 무료 다이아몬드/코인 획득, 일일 무료 가차 1회
- **게임 패스**: 구매 시 레벨업에 따라 다이아몬드 획득
- **일간/주간/월간 할인팩**: IAP 전용
- **가차 시스템**: Spring Carnival 등 특정 목표를 선택하는 고가치 가차 (3티켓 = 풀 풀링)
- **캐릭터 직접 구매**: $19.99 ~ $29.99
- **시즌 패스**: 유료 트랙에서 프리미엄 보상

### 7. 게임 모드

- **메인 챕터**: 기본 생존 모드
- **도전 (챌린지)**: 강화된 난이도 + 보상
- **온라인 도전**: 랭킹 경쟁 + 친구 기능
- **협동 전투**: 특공대 동조 기능 (멀티플레이)
- **압도 모드**: 올 맥스 상태 특수 플레이
- **한정 이벤트**: 특공대 달리기, 클로버 동화 나라 등 시즌별 이벤트 콘텐츠

### [게임 디자이너] 코어 루프 & MVP 시스템 설계

**MVP에 포함할 핵심 시스템 (우선순위 순):**

1. **이동 시스템**: 싱글 조이스틱, 8방향 이동. 이것만으로 조작 끝
2. **자동공격**: 가장 가까운 적에게 기본 무기 자동 발사. 무기마다 공격 패턴(직선탄, 회전, 범위 등) 차별화
3. **경험치 & 레벨업**: 적 처치 → 보석 드롭 → 자석 흡수 → 게이지 → 레벨업 시 3택 스킬 선택
4. **스킬 시스템**: 액티브 스킬 6슬롯 + 패시브 스킬 6슬롯. 동일 스킬 재선택 시 레벨업(최대 5)
5. **스킬 진화(돌파)**: 특정 액티브+패시브 조합이 맥스일 때 진화. MVP에서 3~4개 조합이면 충분
6. **웨이브 & 보스**: 시간 경과에 따라 적 밀도/종류 증가, 2~3분 간격 보스 등장

**MVP에서 빼도 되는 것:**
- 장비 시스템 (메타 진행) → 코어 루프 검증 후 추가
- 캐릭터 다중 선택 → 기본 캐릭터 1명으로 시작
- 챕터/스테이지 시스템 → 단일 맵에서 생존 모드만
- 방치형(퀵 패트롤) → 후순위

**프로토타입 마일스톤 제안:**
```
M1: 이동 + 자동공격 + 적 스폰 (재미 검증의 핵심)
M2: 경험치 + 레벨업 스킬선택 UI
M3: 스킬 5~6종 구현 + 패시브 3~4종
M4: 스킬 진화 + 보스
M5: 단일 맵 완성 (8분 세션, 웨이브 테이블)
```
